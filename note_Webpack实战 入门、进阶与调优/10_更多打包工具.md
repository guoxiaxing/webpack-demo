### 下面两个都只需要全局安装即可

#### Roolup

更专注于 JavaScript 打包

不会添加无用代码

tree shaking 也是基于 es6 module 进行静态分析

可以选择输出模块的模式 output.format （cjs amd esm life umd system）

#### Parcel

Parcel 在打包速度的优化上主要做了 3 件事：

· 利用 worker 来并行执行任务；webpack 也有

· 文件系统缓存；webpack 也有

· 资源编译处理流程优化。

- 零配置

1. 可以以 html 文件作为入口寻找依赖

2. 自动生成 source map 文件

3. 自动为打包后的文件(js)的文件名添加 hash 值

4. 打包后的文件也进行了压缩

#### webAssembly

性能可以媲美于原生，另外像 C 和 Java 等语言都可以编译为 WebAssembly 在现代浏览器上运行。

#### tree shaking 将 mode 改为 production 能生效；

webpack 为什么不学习 parcel 编译做法？

目前 webpack 已经支持了 AST；
